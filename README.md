Teamwork Android App
================================

An Android Teamwork app project with the following:

- AndroidX
- Android Google Material 
- Navigation
- Architecture Components (LiveData, ViewModel)
- Coroutines
- OkHttp3
- Picasso
- Butterknife
- Preferences

### Language
- Kotlin
- Java

### Architecture
- App 
- Domain
- Model
- Network

### Architecture integration
- App <-> Domain <-> Model
- Domain <-> Network <-> Model
- Network <-> Model
